# NDA Policy Rules (Mock)

1. **Term length**: Confidentiality obligations lasting longer than 5 years must be flagged.
2. **Mutuality**: One-sided (non-mutual) NDAs where only one party is bound must be flagged
   for legal review.
3. **Residuals clause**: Any clause allowing the receiving party to retain and use
   information from memory ("residuals") must be flagged — high leak risk.
4. **Governing law**: Same rule as contracts — must specify a mutually agreed jurisdiction.
